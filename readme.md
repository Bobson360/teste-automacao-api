## Objetivo

Automatizar um teste de API no sistema Reqres.in
 
#### Cenário a ser automatizado

1. Executar a request GET https://reqres.in/api/users?page=2
2. Validar que o status code é igual a 200

## Pré-Condições

* java
* rest-assured
* TestNg ou Junit
